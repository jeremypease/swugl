How to subscribe

Apple Calendar (iOS / macOS)
  1. Copy the URL above.
  2. Open Calendar → File → New Calendar Subscription…
  3. Paste the URL and click Subscribe.
  4. Set Auto-refresh to "Every hour" and tap OK.
Google Calendar
  1. Copy the URL above.
  2. Open Google Calendar → click + next to "Other calendars".
  3. Choose From URL, paste the URL, and click Add calendar.
Outlook
  1. Copy the URL above.
  2. Go to Calendar → Add calendar → Subscribe from web.
  3. Paste the URL and click Import.
